# 🎮 Juego del Ahorcado - Proyecto de Aprendizaje y Refactoring

¡Bienvenido al repositorio del clásico juego del Ahorcado!

## 🎯 Objetivo del Proyecto

Este es uno de mis primeros proyectos complejos en **Python**. Más allá de construir un juego funcional, el objetivo principal de este repositorio es utilizarlo como un entorno de práctica para mejorar mis habilidades de desarrollo y diseño de software. 

A través de las diferentes versiones de este proyecto, busco aprender e implementar:

- **Refactoring:** Evolucionar el código desde un script simple y monolítico hacia una arquitectura limpia, estructurada y mantenible.
- **Principios SOLID:** Diseñar clases y módulos que cumplan con las buenas prácticas de la programación orientada a objetos (POO) para crear un sistema robusto y escalable.
- **Interfaz Gráfica (GUI):** Transformar la interacción del juego, pasando de la consola de comandos (CLI) a una interfaz visual atractiva y amigable para el usuario.
- **Patrones de Diseño:** (En etapas futuras) Identificar problemas comunes en la estructura del código y aplicar patrones de diseño para resolverlos de manera elegante.

## 🚀 Hoja de Ruta (Roadmap)

El desarrollo está planteado como un proceso iterativo, partiendo de lo más sencillo para ir aumentando la complejidad técnica:

1. **Fase 1:** Lógica básica funcional en consola (CLI).
2. **Fase 2:** Refactorización inicial a Programación Orientada a Objetos (POO).
3. **Fase 3:** Refactorización avanzada aplicando principios SOLID.
4. **Fase 4:** Implementación de la Interfaz Gráfica de Usuario (GUI).
5. **Fase 5:** Integración de Patrones de Diseño.
